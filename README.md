# mark2web

Markdown to webpage link

## Usage

```bash
$ mark2web test.md
https://mark2web.test/aa32d8f230ef9d44c3a7acb55b572c8599502701
$ mark2web /tmp/session/test.md
https://mark2web.test/aa32d8f230ef9d44c3a7acb55b572c8599502701
```

## Install

### Go

```bash
$ go get github.com/thealamu/mark2web
```
